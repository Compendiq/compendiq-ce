Open source. Self-hosted. Connects to your Confluence Data Center, syncs your knowledge base, and lets you ask questions, improve articles, and generate new ones using the LLM of your choice.

Self-host for free GitHub ★ —

RAG-powered Q&A

Hybrid search across every synced page — pgvector cosine + full-text + RRF re-ranking. Ask a question, get an answer grounded in your real Confluence content, with citations.

Your LLM, your rules

Ollama by default (fully local, zero API cost). Or bring your own OpenAI-compatible API — OpenAI, Azure OpenAI, LM Studio, vLLM, llama.cpp, LocalAI. Per-user or server-wide.

Confluence DC-native

XHTML storage format bidirectional sync. Draw.io diagrams render. Macros round-trip. Written for Confluence DC 8.x and 9.x on-premise — not a Cloud-only rewrite.

Install on your own machine

One command. Generates secrets, pulls images, opens the first-run wizard.

curl -fsSL https://raw.githubusercontent.com/Compendiq/compendiq-ce/main/scripts/install.sh | bash

Tested on macOS 14 (Sonoma) and Rocky Linux 10. Ubuntu, Debian, and Windows + WSL2 coverage lands in Phase 2. Requires Docker Desktop on macOS or Docker Engine on Linux.
