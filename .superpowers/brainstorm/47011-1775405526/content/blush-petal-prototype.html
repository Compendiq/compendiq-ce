Getting Started with Confluence Sync

Compendiq connects to your Confluence Data Center instance and synchronizes spaces, pages, and attachments. Articles are converted from XHTML to Markdown for AI processing via the confluenceToHtml() pipeline.

Pro tip: Start with a small space to verify your PAT configuration before enabling full sync across all spaces.

ℹ️

Confluence PATs are encrypted with AES-256-GCM at rest. They are never sent to the frontend or exposed in API responses.

✅

Use the SYNC_INTERVAL_MIN environment variable to control how often automatic sync runs.

// Configure sync for a space const config = { spaceKey: 'ENG', syncInterval: 15, // minutes includeAttachments: true, embeddingModel: 'nomic-embed-text', }; const result = await syncService.startSync(config); console.log(`Synced ${result.pageCount} pages`);
Environment Variable Default Description
SYNC_INTERVAL_MIN 15 Polling interval in minutes
CONFLUENCE_VERIFY_SSL true TLS verification for Confluence API
ATTACHMENTS_DIR data/attachments Local cache directory for attachments
⚠️

Never set CONFLUENCE_VERIFY_SSL=false in production. This disables TLS certificate verification and exposes your PAT to MITM attacks.
